# TestAutomation

Installation and Execution Guide

1. Add a Java JDK of any version in tle local computer.
2. Download Eclipse.
3. Import the project from the GitHub.
4. Update the Maven project, so the dependencies gets downloaded from the central repository.
5. Allow some time to execute until the dependencies to get downloaded.
6. Run the maven project.

Task 1 description:
1. Implemented the project with BDD Framework with TestNG.
2. Feature files are located in test/resources folder.
3. Incorporated Test Reports using extent-reports.
4. Added screenshot functionality after eact Test Steps gets evaluated.
